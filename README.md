# MintSells

## 简介

基于SpringBoot的轻量级、适合新手入门、针对微信小应用的买卖家后台快速开发系统。

## 特点

分布式系统设计、Redis缓存功能、秒杀活动（Redis分布式锁设计）、微信支付退款快速接入、微信登录扫码、WebSocket前后端通信、FreeMarker模板前端生成
